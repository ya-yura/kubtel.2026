# Kubtel.ru: QA визуальной системы этапа 3

Дата проверки: 2026-05-07.

Проверка закрывает техническую часть визуального этапа: адаптивность, отсутствие критичных layout-дефектов, базовую доступность интерактивных элементов и контраст ключевых цветовых пар.

## Метод

Использован bundled Playwright из локального Codex runtime без установки новых зависимостей.

Проверенные страницы:

- `/`;
- `/tariffs/`;
- `/connect/`;
- `/support/`;
- `/contacts/`;
- `/about/`.

Проверенные viewport:

- mobile: `390x844`;
- desktop: `1440x1000`.

## Исправление в ходе QA

Темный блок метрик в hero был переведен с полупрозрачного внутреннего фона на solid dark `#2a2f36`. Визуально слой остался технологичным и спокойным, но автоматическая проверка контраста теперь корректно видит фактическую цветовую пару.

## Ограничения

Проверка закрывает интерфейсный слой текущего прототипа. Финальный визуальный аудит нужно повторить после появления подтвержденных бренд-материалов Kubtel: логотипа, фотографий, иконок, реальных контактов и офисных данных.
